## ✨ Features

* 🔐 **Secure Authentication**
  Access the controller using a unique API key, ensuring only authorized users can manage the bot.

* 📊 **Real-Time Status Monitoring**
  Instantly view whether your AI assistant is active or disabled.

* 🎛️ **Remote Toggle Control**
  Enable or disable AI auto-replies with a single tap from anywhere.

* 🌐 **Server Connectivity Check**
  Monitor backend server availability and connection health.

* 🛡️ **Secure Local Storage**
  API keys are securely stored using `flutter_secure_storage`.

* 🎨 **Modern UI/UX**
  Clean Material Design interface enhanced with smooth Lottie animations.

## 📦 Getting Started

### Prerequisites

* Flutter SDK (Latest Stable)
* Android Studio or VS Code
* Flutter Extension
* Valid API Key

### Installation

#### 1️⃣ Clone the Repository

```bash
git clone https://github.com/ars2k03/AI-Controller.git
cd AI-Controller
```

#### 2️⃣ Install Dependencies

```bash
flutter pub get
```

#### 3️⃣ Run the Application

```bash
flutter run
```
